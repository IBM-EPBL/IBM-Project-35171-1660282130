This project aims to aid the deaf-mute by creation of a new system that helps convert sign language to text and speech for easier communication with audience. The system consists of a gesture recognizer hand-glove which converts gestures into electrical signals using flex sensors. These electrical signals are then processed using an Arduino microcontroller and a Python-based backend for text-to-speech conversion. The glove includes two modes of operation â€“ phrase fetch mode and letter fetch mode. The phrase fetch mode speaks out words at once, while the letter fetch mode speaks out individual letters. This project forms a base infrastructure which can later be augmented with addition of different Sign Languages and integrating with other hearing impaired aid systems.
